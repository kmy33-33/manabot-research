# ManaBot Research

ManaBot Research is the open-source research and infrastructure layer behind ManaBot — an AI-powered trading intelligence system focused on:

- Multi-timeframe market analysis
- AI reasoning and conversational analysis
- Macro awareness
- Reinforcement learning
- Long-term market memory
- Correlation analysis
- Market regime detection
- Autonomous trading infrastructure

## Purpose

This repository contains:
- Research experiments
- Market data tooling
- AI infrastructure
- Learning systems
- Backtesting frameworks
- Knowledge ingestion systems
- Documentation and architecture

Commercial execution infrastructure and proprietary trading systems remain private.
